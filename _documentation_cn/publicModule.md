    | --- | --- |
    | SUBMIT | 提现: 未冻结 |
    | REVIEW | 提现: 已冻结,待审核 |
    | AUDITED | 提现: 已审核,发送钱包,待上链 |

    | SUCCESS | 完成 |
    | FAIL | 失败 |
    | CANCEL | 已取消 |

    | REJECTED | 下单失败 |
